## Architectural seams enforced by the tech-lead

These are non-negotiable. See `.claude/agents/tech-lead.md` for the full list. Summary:

1. Package boundaries (KAI-236) — depguard linter enforced
2. Connect-Go `.proto` is the only inter-role contract
3. `IdentityProvider` interface is the identity firewall
4. Multi-tenant isolation chaos test (KAI-235) on every cloud API PR
5. Stream URL minting is asymmetric (sign cloud-side, verify local)
6. Recording never stops — fail closed for security, fail open for recording
7. Face recognition ships with full EU AI Act package
8. Every customer-visible string goes through i18n override
9. Build for multi-region, ship single-region
10. Do not modify `mediamtx.yml` runtime settings to pass tests
